# Webサイト開発者用編集環境

このフォルダは、Webサイト制作の第1段階として使う開発者用ワークスペースです。

## 開く場所

VS Codeで以下のフォルダを開きます。

```text
C:\Users\kipep\website-workspace
```

## 主なファイル

- `index.html`: ページ構造と表示内容
- `styles.css`: 色、余白、レイアウト、レスポンシブ対応
- `script.js`: ブラウザ上の簡易編集UI
- `website-workspace.code-workspace`: VS Codeワークスペース設定

## 確認方法

`index.html` をブラウザで開くと確認できます。

Live Server拡張機能を使う場合は、`index.html` を右クリックして `Open with Live Server` を選びます。

## 次に決めること

- サイト名
- サイトの目的
- 掲載したい文章
- 掲載したい画像やロゴ
- 問い合わせ先やフォームの有無
